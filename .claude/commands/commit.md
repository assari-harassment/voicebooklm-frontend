---
description: 変更内容から日本語コミットメッセージを生成してコミット
---

以下のコマンドで現在の変更とブランチ名を確認してください：

```bash
git branch --show-current
git status
git diff --staged
git diff
```

確認した変更内容をもとに、以下のルールでコミットしてください：

## ルール

- 日本語で一文、簡潔に書く
- 改行しない
- 何を変更したかが分かるように書く
- **最後にブランチ名から取得したissue番号を `#番号` の形式で付ける**
  - 例: `feature/42` → `#42`
  - 例: `fix/123` → `#123`

## コミットメッセージの形式

```
変更内容の説明 #issue番号
```

例: `ログイン機能を追加 #42`

## 手順

1. 必要なファイルを `git add` する
2. コミットメッセージを提案して確認を取る
3. `git commit -m "メッセージ"` でコミット
